Session Buddy JSON to HTML Converter
===
This tool transforms a JSON file exported from the Session Buddy Chrome extension into a single HTML page. It generates a convenient list of clickable links to stored websites, all while working locallyâ€”no server uploads are required.

How to Use
====
1. Open the HTML file in your web browser.
1. Load your Session Buddy JSON file using the provided interface.
1. The tool will generate a list of links based on the contents of the JSON file.
1. Save the resulting HTML as a static file for future use.
